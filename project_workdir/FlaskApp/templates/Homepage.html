InvestiGame Prediction Engine


Investors, resellers, and gaming enthusiasts would like to know if a new video game will be commercially sucessful since this information can be used to as a signal for investment decisions and to provide insight into video game performance based on professional reviews and community assessment. Investigame uses Machine Learning tools to calculate and predict how a game that is about to be released will perform on the market using Social Media text analytics, publisher/developer info, and professional reviews for the game. New games that are about to be released are searched for and added each day to the database. Select a game from the dropdown bar below and click Predict to see how this game will perfom.
